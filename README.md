## Technology
- Node

## Requirements
- Node.js
- NPM

## Package installation
Execute the following command on your terminal to install all the needed packages:
```
git clone https://github.com/soumenmaity106/Node-basic-Rest-Api.git

npm install
mongod

node app.js

```
